# Get-EasyAid 🎓

> AI-powered tool to generate personalized financial aid application answers for Coursera courses.

## 📖 About

Get-EasyAid helps students and professionals apply for financial aid on Coursera by generating AI-powered, personalized responses to financial aid application questions. Simply search for your desired course, fill in your details, and get ready-to-use answers tailored to your profile.

## ✨ Features

- 🔍 **Course Search**: Search through Coursera courses by name or university
- 📝 **Personalized Forms**: Fill in your details (student/working status, institute, etc.)
- 🤖 **AI-Generated Answers**: Get two comprehensive responses:
  - Why you're applying for financial aid
  - How the course will help achieve your goals
- 📋 **Copy & Regenerate**: Easily copy responses or regenerate new ones
- 💾 **Offline Support**: Course data cached locally for faster access
- 📱 **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- 🧹 **Clean Code**: Well-structured, maintainable codebase with modern React patterns

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/VoyagerX21/get-easyAid-client
cd get-easyAid-client
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## 📝 Usage

1. **Search**: Enter a course name or university in the search bar
2. **Select**: Choose your desired course from the dropdown
3. **Fill Details**: Complete the form with your personal information
4. **Generate**: Click "Get Result" to generate your financial aid responses
5. **Copy**: Use the copy button to copy responses to your clipboard

## ⚠️ Disclaimer

AI-generated answers may have flaws. Please review and customize the responses before submitting your financial aid application.
